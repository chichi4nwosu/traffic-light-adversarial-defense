# Datasets

This project uses two public traffic light detection datasets.

## Bosch Small Traffic Lights Dataset (BSTLD)

Used for training, adversarial attack evaluation, and adversarial training.

Dataset:
https://hci.iwr.uni-heidelberg.de/node/6132

---

## LISA Traffic Light Dataset

Used to evaluate generalization of attacks and defenses.

Dataset:
https://cvrr.ucsd.edu/LISA/lisa-traffic-sign-dataset.html

---

The datasets are **not included** in this repository because of their size and licensing restrictions.

Please download them separately and update the configuration files accordingly.
